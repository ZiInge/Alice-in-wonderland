# Alice in Wonderland

Alice is an automated hidden garage door for a Roborock Saros 10.

## Project Goals

- Automatic opening when the robot leaves the dock
- Automatic closing when the robot returns
- Position feedback from the actuator
- ESPHome + Home Assistant integration
- Safe operation with calibration and limits

## Hardware

- ESP32-S3 DevKitC-1
- Home Assistant Green
- Actuonix L12-I Linear Actuator
- Mean Well GST60A12-P1J
- 12V → 5V Buck Converter
